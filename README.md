Мини-приложение для ведения заметок.

Приложение поддерживает две функции: add_note и list_notes.
add_note добавляет заметку в БД, пример: 
.../NotesPlus$ python -m src.main add_note "Заметка"
Строка "Заметка" добавляется в БД.
list_notes выводит весь список заметок из БД, пример:
.../NotesPlus$ python -m src.main list_notes
Заметка
